マントとスカートの突き抜け（貫通）の対処方法について

　　モーションによっては、マントとスカートに突き抜け（貫通）が発生しやす
くなるものがあります。このようなモーションの場合、モーションの初期姿勢
を５フレーム目位にコピーして、１フレーム目を初期状態（直立状態）に戻し、
５フレーム目からセンター、ＩＫ、すべての親等だけを１フレーム目にコピー
した後で、モーションを再生してみてください。
全てのモーションで確かめた訳ではありませんが、以上の方法で、マントと
スカートの突き抜け（貫通）が緩和される場合があります。
